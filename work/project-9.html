Dash AR

Project Overview
By using Virtual Reality or Augmented Reality, create a product that can encourage continued healthy living. The end result was an Augmented Reality program that would allow a user to make the practice of running or walking more enjoyable by transforming it into a game with features allowing you to race your friends, run from zombies, or simply race past versions of yourself.
